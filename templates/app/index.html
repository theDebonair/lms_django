OUR AIM

The main objective of this project is to manage the details of the students, books and the issued books efficiently. This project helps to keep all the records of the library. Students can register themselves to this system. The admin can then issue them with any book they want to read for a period of 7 days. The students are requested to return it within 7 days of time or else there is a ₹5 fine/day till the book is being returned. After returning, they can re-issue a new book if the student wants.
